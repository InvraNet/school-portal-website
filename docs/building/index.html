Building the School Portal

This guide provides an overview of the prerequisites required to build the School Portal application. Before you start the build process, make sure you have the following tools installed:

Prerequisites

1. Git

Git is a version control system that you will need to clone the source code repository. You can download and install Git from git-scm.com.

2. Node.js

Node.js is a runtime environment required for running build tools. It is usually bundled with the application, but if not, you can install it from nodejs.org.

3. Yarn

Yarn is a package manager used for managing dependencies. It simplifies the process of installing and managing packages. Install Yarn from yarnpkg.com.

Ensure you have all these tools installed and properly configured before starting the build process. They are essential for compiling and running the School Portal application successfully.

Building School Portal on Windows

Follow these steps to build the app for Windows using Electron Forge.

Step 1: Install Node.js

choco install nodejs

Step 2: Install Yarnpkg

corepack enable

Step 3: Clone the Repository

git clone https://github.com/invranet/school-portal

Step 4: Install All Packages

Reload your terminal to ensure all environment variables are set properly.

yarn

Step 5: Build the App

yarn dist:win

Your app will be available in the dist/ folder.
